The Storm-DFT tool has been validated against multiple fault trees in a DFT benchmark suite FFORT – a fault tree forest – maintained by Twente University the Netherlands. FFORT is a collection of fault trees gathered from scientific literature and open industrial reports. One can find the analysis results for the fault trees verifiable by our tool in the suite in comparison with other tools (if any).
